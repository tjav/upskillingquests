# infra-basics-14

## Description

As a cloud engineer, we are going to do a lot of testing with Azure Services. Many of the services might be only available via private connections. Hence, we want to have a script that deploys windows 10 VM in a Network which can be reused when testing to Azure Services.

## Outcome

- create architecture (draw.io/ppt/visio) of the solution.
- Show in the architecture how the networking works and how the user will connect.
