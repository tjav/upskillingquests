# infra-basics-18

## Description

As company data is saved in this case a storage account. The company want to gain insights on the data in azure. 

For this they want to take a look into Azure Purview. Hence, they want you to create a purview account and make sure you can connect to the portal via a private endpoint.

## Outcome

- Design the solutions
- Deploy a Purview account.
- Make sure to create a private endpoint for the purview portal.
